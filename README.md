A Vinheria Agnello é uma aplicação web de catálogo e venda de vinhos, com foco em experiência visual premium e uso simples em computador e celular.
O sistema permite navegar por categorias, adicionar produtos ao carrinho e finalizar compras com controle de sessão de usuário.

Principais pontos:

Catálogo organizado por tipo de vinho (brancos, rosé, tintos e espumantes).
Carrinho com ajuste de quantidade, remoção de itens e total automático.
Painel administrativo para cadastrar, editar e excluir vinhos.
Upload de imagens e gerenciamento de disponibilidade dos produtos.
Sincronização de catálogo entre dispositivos (desktop e mobile), garantindo dados consistentes.
Tecnologias usadas:

HTML, CSS e JavaScript no front-end.
APIs serverless na Vercel com Vercel Blob para armazenamento.
